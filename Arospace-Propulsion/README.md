# Vinci Second Stage Propulsion System

Technical overview of the Vinci engine, the restartable cryogenic upper-stage propulsion system used on Ariane 6.

The project was developed during the Bachelor's Degree in Aerospace Engineering at Politecnico di Milano. It focuses on the main subsystems of the engine, including the feed system, expander cycle, turbopumps, regenerative cooling, injection, ignition system, combustion chamber, and nozzle section.

## Main Topics

- Cryogenic rocket propulsion
- Ariane 6 upper stage architecture
- Vinci engine technical overview
- LH2/LOX feed system
- Expander cycle
- Turbopump performance
- Regenerative cooling
- Injection and ignition systems
- Combustion chamber and nozzle flow
