# Introduction to Mission Analysis Project

This repository contains the final project developed for the *Introduction to Mission Analysis* course, part of the Bachelor's Degree in Aerospace Engineering at Politecnico di Milano.

The project focuses on the analysis and comparison of different orbital transfer strategies between two assigned orbits. The main objective is to evaluate the trade-off between transfer time and fuel cost, expressed through the total velocity increment, Δv.

The work includes the characterization of the initial and final orbits, the implementation of different transfer approaches, and the numerical comparison of their performance.

## Main Topics

- Orbital mechanics
- Keplerian orbital elements
- Cartesian-to-Keplerian conversion
- Orbit transfer maneuvers
- Plane change maneuvers
- Periapsis rotation
- Direct transfer trajectories
- Bielliptical transfers
- Δv and time-of-flight estimation
- MATLAB-based orbital analysis
