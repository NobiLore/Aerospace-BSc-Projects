# Aerospace BE Projects

Selected academic projects developed during my Bachelor's Degree in Aerospace Engineering at Politecnico di Milano.

## Projects

### Introduction to Mission Analysis

Orbital transfer analysis project comparing different maneuver strategies in terms of total Δv and transfer time.

### Aerospace Propulsion

Technical overview of the Vinci cryogenic upper-stage propulsion system used on Ariane 6, covering the main engine subsystems and preliminary engineering analyses.
